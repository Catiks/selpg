# 通过golang实现简单的命令行实用程序

[GitHub 项目地址](https://github.com/Catiks/selpg)

## 程序运行

> 找到selpg可执行文件，命令行中输入./selpg 并加上相应的参数，就能执行

## 参数说明
> -s起始页号 这个参数指明要输出（打印）的页的起始页号，当然，这个页号必须是有效页号，不然会出错
> -e结束页号 这个参数与-s对应，指明要输出（打印）的页的结束页号
> -s起始页号 和-e结束页好 这两个都是必要参数，不能忽略的，所以程序运行的过程中，必须要有两个参数
> 可选参数：
 > -f 这个参数提供了一个选项，就是对于每页所包含的行数的选择 添加-f之后，就会让每一个\n（换行符）变成一个页结束的标志，也就是一行表示一页
 > -l行数 这个可选项提供可选的每一页所表示的行数，比如-l10就表示每十行代表一个页

## 文件说明
> input 这个文件是一个样例的模拟输入文件，一个500+行，基本可以保证了调试
> output 这个文件是输出样本文件，对于输出到文件的运行，输出结果都写入这个文件
> errorlog 这个文件是错误记录日志文件，每次程序运行出现的错误，都会记录到这个文件中，方便查看
